rmss 設計思考メモ
= これは何
WMO/JMA ソケット手順などの電文交換プログラムを ruby で実装するもの。

WMO 国際通信関係者間ではこの種のプログラムを message switching system という
（日本気象庁では「アデスのようなもの」と言わないと通じない）のが名前の由来。
さしあたり ruby で実装するが、いずれ C で書くこともあるかもしれないので、
区別できるように rmss と命名しておく。
